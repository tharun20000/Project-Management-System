# DevSync Project
Welcome to DevSync - a project management app designed to help you manage your projects more efficiently.

## Features
- Project management
- Team collaboration
- Community building
- Time tracking

## Benefits
- Increased productivity
- Improved communication
- Better project outcomes
- Increased collaboration
- Networking opportunities


## Technologies Used
- React Js
- NodeJs
- Express Js
- Mongo Db
- Styled Components
- Material-UI
- Nodemailer

## Environment Setup
- Create a `.env` file in `server/` (do not commit it).
- Use `server/.env.example` as a template and fill real values.
- Required server variables:
  - `PORT`: Server port (default 8700)
  - `MONGO_URL`: MongoDB connection string
  - `JWT`: JWT signing secret
  - `USER_VERIFICATION_TOKEN_SECRET`: Email verification token secret
  - `EMAIL`: Sender email address
  - `EMAIL_USERNAME`: SMTP username (e.g., Gmail address)
  - `EMAIL_PASSWORD`: SMTP app password
  - `URL`: Base URL for client links (e.g., http://localhost:5173)

### Running the server
1. Install dependencies:
	- `cd server`
	- `npm install`
2. Copy env template:
	- `cp .env.example .env` (then edit values)
3. Start:
	- `npm run dev`
